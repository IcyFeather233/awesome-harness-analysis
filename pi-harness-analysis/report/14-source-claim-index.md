- 点击源码链接时应核对固定 commit，不要用当前默认分支替代版本事实。

- 完整 conditions、counterevidence、coverage refs 和 record schema 以 [`claims.jsonl`](../evidence/claims.jsonl) 与 [`observations.jsonl`](../evidence/observations.jsonl) 为准。
